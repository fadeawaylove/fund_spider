# 总体流程  
- 爬虫或者手动方式获取资料，保存到文件仓库（fastdfs）
- 手动筛选有价值的资料
- 对文件进行加密压缩，然后上传到百度云
- 将文件概览图片和解压密码上传到知识付费平台
- 将资料的百度云链提交到
